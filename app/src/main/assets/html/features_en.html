TVHClient is free of charge and free of advertising. This will remain so in the future. A purchase of the application offers however the following further functions:
  • Integrated Player
    Watch live TV or recordings on the device with the integrated player. The HTSP protocol is used.
  • Playback on the Chromecast
    Send live TV or recordings to a Chromecast. The server must be able to transcode the videos in real time.
  • Download recordings
  • Edit scheduled and running recordings
  • Edit series recordings
  • Create and edit timer recordings
    Determine yourself which programs are recorded when and how. Define these recordings by parameters like start and end time, day of recording and more. (At least API version 18)
  • Enhanced series recording
    Fine tune your series recordings! Define more customized series recordings by specifying additional parameters like start and stop time, duration, day of week to record and more. Additionally you can edit existing series recordings.
    (Minimum required API version is 13)
  • Notifications
    Notifies the user that a scheduled recording has been started or stopped, even if TVHClient is not running.
  • Menu 'Record and edit program'.
    Offers the possibility to directly edit a program after it has been scheduled.
  • Select multiple channel tags
  • Jump to a defined timeframe in the EPG
  • Search program information on Youtube and Google
  • Search as you type
    In the search screen the search will start automatically when 3 or more characters are entered.
  • Wakeup Server via Wake On LAN
