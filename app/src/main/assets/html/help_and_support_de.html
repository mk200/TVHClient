Wiedergabe
Externe Player

Falls der externe Player gestartet wurde, aber kein Video erscheint, prüfe folgende Punkte:

  • Das gewählte Wiedergabe-Profil ist nicht geeignet. Empfohlen wird 'matroska'.
  • Der Streaming-Port (Default ist 9981) ist nicht korrekt.
  • Der Player kann den Server über den Streaming-Port ist erreichen.
  • Der Upload der Daten vom Server ist zu langsam.
Interner Player

Falls der interne Player gestartet wurde, aber kein Video erscheint, prüfe folgende Punkte:

  • Das gewählte Wiedergabe-Profil ist nicht geeignet. Empfohlen wird 'htsp'.
  • Der Upload der Daten vom Server ist zu langsam.
Casting
Anforderungen
  • Du benutzt die freigeschaltete Version von TVHClient.
  • Dein Server unterstützt Transcodierung.
  • Transcodierung, insbesondere VAAPI hardwaregestützte Transcodierung, funktioniert am besten mit den neuesten instabilen TVH 4.3 Versionen.
Einrichtung der Transkodierung

Für maximale Kompatibilität verwende ein Stream-Profil mit den folgenden Einstellungen:

  • Container: av-lib/MP4 (Beispiel)
  • Video codec: H.264, High profile (Beispiel)
  • Audio codec: AAC LC profile (Beispiel)
TVHClient einrichten
  • Verwende entweder eine IP-Adresse oder einen DNS-Namen für die Verbindung. Vor dem Start des Castings wird der DNS-Name in eine IP-Adresse umgewandelt.
  • Wähle das entsprechende Casting-Wiedergabeprofil aus.
